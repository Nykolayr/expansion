Description

Expansion is a portrait space RTS on a grid of nodes. Capture bases, send fleets, grow nodes, and make tactical calls every battle.

The Classic campaign is a chain of missions with rising difficulty. Core content ships in the app; new missions and layout updates download from the server — the campaign grows without a store reinstall.

Features

  • Capture and hold bases on the space map
  • Send fleets between nodes along straight lines
  • In-battle tactical upgrades for resources
  • Persistent meta upgrades between missions
  • Tutorial from mission one and in-game help
  • Guest mode or account with progress sync
