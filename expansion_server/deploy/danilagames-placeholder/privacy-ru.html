Политика конфиденциальности

Приложение Expansion (Экспансия) · com.ryjovs.expansion · обновлено 15 июля 2026 г.

Настоящая политика описывает, какие данные обрабатываются в мобильной игре Expansion (далее — «Приложение»), и с какой целью. Оператор: разработчик проекта Danila Games (danilagames.ru).

1. Какие данные мы собираем

  • Гостевой режим: прогресс кампании, настройки и идентификатор гостя хранятся локально на устройстве (SharedPreferences, SQLite).
  • Аккаунт (по желанию): email, ник, имя (необязательно), хеш пароля на сервере, игровой прогресс, очки, мета-апгрейды.
  • Токены входа хранятся в защищённом хранилище устройства (Secure Storage), не в открытом виде.
  • Обратная связь: текст сообщения и email (для гостя — указанный в форме; для аккаунта — из профиля).
  • Донаты: код платежа (EXP-XXXXXX), сумма и статус на сервере. Данные банковской карты в Приложении не собираются — оплата через страницу банка (СБП).
  • Реклама (Yandex Mobile Ads): рекламный SDK может получать технические данные устройства и идентификаторы в соответствии с политикой Яндекса.
  • Технические данные: запросы к API (IP, user-agent) в стандартных логах сервера для работы сервиса и безопасности.

2. Зачем это нужно

  • запуск игры, сохранение прогресса, синхронизация между устройствами;
  • регистрация, вход, восстановление пароля;
  • показ рекламы и опциональная награда за просмотр (rewarded);
  • обработка донатов и отключение рекламы после подтверждения оплаты;
  • поддержка пользователей и улучшение игры.

3. Где хранятся данные

Данные аккаунта и прогресса — на сервере API (VPS, РФ). Локальный кэш контента кампании — в SQLite на устройстве. Гостевой прогресс без аккаунта при удалении Приложения теряется.

4. Передача третьим лицам

Мы не продаём персональные данные. Передача возможна только:

  • рекламному партнёру (Яндекс) — для показа рекламы;
  • банку / платёжной странице — при добровольной оплате доната пользователем;
  • по требованию закона РФ.

5. Ваши права

  • играть без регистрации (гостевой режим);
  • удалить аккаунт и данные — см. инструкцию по удалению аккаунта;
  • написать нам по контактам на странице Поддержка.

6. Дети

Приложение не предназначено для детей младше 6 лет. Мы сознательно не собираем данные детей без согласия родителей.

7. Изменения

Актуальная версия всегда на этой странице. При существенных изменениях дата обновления будет изменена.
