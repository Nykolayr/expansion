# Expansion — монорепозиторий

Космическая стратегия (новая кодовая база на [digitalsquare](https://github.com/Nykolayr/expansion)).

## Структура

| Каталог | Описание |
|---------|----------|
| [`expansion_app/`](expansion_app/) | Flutter-клиент (iOS, Android) |
| [`expansion_server/`](expansion_server/) | Backend API (в разработке) |
| [`expansion_admin/`](expansion_admin/) | Админ-панель (в разработке) |

## Быстрый старт (клиент)

```bash
cd expansion_app
flutter pub get
flutter gen-l10n
flutter run
```

Спецификация игры: [`expansion_app/docs/project-specs.md`](expansion_app/docs/project-specs.md).
