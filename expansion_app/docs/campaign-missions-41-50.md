# Кампания Classic: миссии 41–50 (туманность Андромеды)

> Статус: **утверждено** (2026-07-17).  
> Публикация: OTA — [`docs/campaign-content-ota.md`](../../docs/campaign-content-ota.md).  
> Новых hazard/variant **нет** — ротация существующих + геометрия.

## Принципы

- После м40 («финал» первой сотни) — **передышка** на м41, затем разнообразие раскладок.
- Комбо hazards только на **49–50** (boss-lite туманности).
- Темп AI: туманность Андромеда (`×1.15`).

Генератор: `expansion_server/scripts/generate-missions-41-50.js`.
